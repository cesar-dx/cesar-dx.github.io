Featured projects

Optimisation Project

Built a two-stage mixed-integer optimization model to improve water access in the Democratic Republic of Congo by optimally repairing and placing wells under budget constraints. The model maximizes population coverage first, then minimizes average walking distance using population-aware demand grids (WorldPop) and real infrastructure data (WPDx).

Report

Machine Learning Project

Built an interpretable predictive–prescriptive system for MLB betting by combining team performance, starting pitcher trends, and contextual game factors. Applied optimization-based tree models to translate win-probability forecasts into data-driven betting decisions and evaluated performance against market baselines.

Report

Hands-on Deep Learning

Built a multimodal resale-price predictor for the MIT Sloan Buy & Sell WhatsApp marketplace, suggesting a fair reference price from a listing’s image + short, noisy text. Used ViT + BERT embeddings with transfer learning (pretrained on an Amazon multimodal dataset, then fine-tuned on a curated WhatsApp dataset) to outperform unimodal baselines.

Report

Analytics Lab Project

Built an end-to-end ML pipeline to forecast quarterly NAV changes for private equity funds using historical cash flows, valuation dynamics, and public-market/sector signals. Engineered leak-free features and evaluated accuracy with robust error metrics (incl. tail performance), delivering an explainable, sponsor-ready baseline and benchmarking framework.
